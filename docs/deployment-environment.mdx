## Environment variables

These are the environment variables set in the Shuttle runtime container.
Check for `SHUTTLE=true` or use [the shuttle feature flag](./project-structure#cargo-feature-flags) for custom behavior when running on Shuttle.

```bash
SHUTTLE="true"
SHUTTLE_PROJECT_ID="<project id>"
SHUTTLE_PROJECT_NAME="<project name>"
RUST_LOG="INFO"
```
